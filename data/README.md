# Data directory

This directory contains lightweight example data and placeholders used by the MVP version of the project.

## Structure

- `resumes/` — small example resume tables for English and Russian
- `raw/` — placeholder for benchmark downloads and raw resources

## Notes

The full project may rely on external benchmark datasets downloaded separately through `setup_data.sh` or manual setup instructions.

The included CSV files are not intended to replace benchmark datasets. They are provided to:
- document the expected data format
- make the repository easier to understand
- support demonstration and testing of hiring-related logic
